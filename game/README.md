# Harvest Riches

An exciting slot machine game that combines classic casino entertainment with modern design and easy operation. Immerse yourself in a world of symbols, paylines, and thrilling winning opportunities. This game offers hours of fun for casual players and slot enthusiasts alike.

## Installation

1.  Upload the archive to your server.
2.  If necessary, adjust the configuration file `config/config.json` to modify settings such as bet limits.
3.  The game can be accessed via the URL with the additional parameter `?lang=`, to select the desired language. The following languages are supported:

    * `de` (German)
    * `en` (English)
    * `es` (Spanish)
    * `fr` (French)
    * `pt` (Portuguese)
    * `ru` (Russian)
    * `zh` (Chinese)
    * `hi` (Hindi)
    * `ar` (Arabic)
    * `bn` (Bengali)
    * `ur` (Urdu)
    * `id` (Indonesian)
    * `ja` (Japanese)

    Example: `your-domain.com/path-to-game/?lang=en`

## Note

This game is designed as a standalone application. For integration into existing systems, adjustments can be made in the `/classes/Api.php` file.
